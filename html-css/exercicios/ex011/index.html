Imagens Dinâmicas

Tente abrir este site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

É muito importante em um site que o tamanho da imagem seja compatível com a largura da página. Caso aconteça de a largura da imagem ser maior do que a largura do site, a página ficará poluída, aumentando a taxa de bounce rate. Para que o site tenha uma imagem que se adapte a qualquer tamanho de tela, basta utilizar a tag "picture"

A tag "picture" faz com que o desenvolvedor consiga ter mais flexibilidade na especificação de recursos de uma imagem.

Dentro da tag "picture", utilize a tag "img" (para carregar a imagem desejada) e a tag "source", tag esta que permite que o desenvolvedor especifique quais imagens serão utilizadas, dependendo da largura que a página estiver, para que o site possa se adaptar.
