Podemos criar também um texto marcado usando a tag "mark".

Texto Grande / Pequeno

Estamos criando um texto grande (com a tag "big", que no momento se encontra obsoleta) e um texto pequeno (com a tag "small") nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Também é possível sublinhar um texto usando a tag "u" (não semântico).

Para inserir coisas do tipo x20+3 (utilizando a tag "sup").

Texto Subscrito

Para inserir coisas do tipo H2O (utilizando a tag "sub").
